Choropleths from the OK Cupid Data Set

The following maps are based on my "OK Cupid" dataset (raw data available here). For those of you unfamiliar with OK Cupid, it is a free online dating site (and a well-designed one, too, I might add!). Although these maps are based on actual data they should be interpreted with a large grain of salt, as the entire data set includes only 500 people, or an average of 10 people per state. Meaning the margin of error is quite large. I may eventually getting around to adding more data, but for now, these are mainly intended for fun and illustrative purposes only. (Note that there are currently no data for Wyoming or Vermont.)

It may also be worth mentioning that this is a sample of users on the OK Cupid website which is almost certainly not representative of each state or the overall population of the country. I will prepare more information about the data set soon.

A "choropleth" is a color-coded map like those below, where the colors correspond to the values of some variable of interest.
