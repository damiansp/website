Machine Learning

This section of the website provides sample walkthroughs of several commonly used machine learning techniques, including:

Principal Components Analysis (PCA)

Principal Components Analysis used to determine if and the extent to which animal species can be identified by their fatty acid composition.

K-Means Clustering

The k-means clustering algorithm was used to resolve and match identities of unknown fatty acids from the output of a gas-liquid chromatograph.
