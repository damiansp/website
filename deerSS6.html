Statistical Analyses (part 5)

Damian Satterthwaite-Phillips

Fetal Size

We will consider fetal size in two different ways. First as the proportion ln(mass) / ln(crLength), recalling from before that this is a constant, regardless of the gestation age, and may be thought of as an idicator of the health or “robustness”, of the fetus. Next, we will look at total fetal mass, that is, the mass of all fetuses for a given mother. This latter metric can be thought of as a metric of the maternal investment for the pregnancy, rather than as a reflection of the health of any single fetus. We refer to these two metrics as “fetal size ratio”, and “total fetal mass” respectively.

Note the left skew to the distribution of fetal size ratios, and that they are not easily normalized. We can consider a linear model, but as above, a GLM with a gamma-distributed response may be more appropriate. We explore both.

For adults and yearlings, the effect of culling intensity of the fetal size ratio is nearly flat. The relationship is nonlinear, but suggests perhaps a slight increase in the ratio (e.g. more mass for the same length, or “more robust”) at the highest intensity levels. Fawns, by contrast, show a noticeable decline in the ratio (“less robust fetuses”) as the culling intensity increases. The exact effect is predicted to depend on the total number of fetuses, as well as the sexes of those fetuses, but the overall pattern is consistent.

Note that the axes in the first of the above two images is are log-log, indicating that the log of the total fetal mass is in an approximately linear relationship with log number of days into the year, but as the histogram (and scatterplot) show, the distribution shows considerable left skew, and the data are not easily normalized. In the previous set of models, the log(mass) to log(length) ratio was constant, with errors randomly distributed, so the date did not have to be taken into consideration. Here however, while the ratio of log(total fetal mass) is in constant proportion to the log(days), the errors are not normally distributed around that constant, as the above graphs indicate. Hence we specifically add the date into the set of predictors, and procede as before.

As before, there may be some slight concern about the non-constant variance in the residuals, so we will revisit this question in some of the other sections as well. For now, though, we again investigate the effect of the culling intensity on the total fetal mass.

predPlot2 = function(mNo, fNo, yr, yl = NULL) {
    f.pred = exp(predict(tfm.glm, 
                         newdata = data.frame(
                             maleNo = mNo, 
                             femaleNo = fNo,
                             age = 'fawn',
                             yearF = yr,
                             intensity = intV,
                             cwd = 'neg',
                             days = 91
                          ), type = 'response') - 2)

    y.pred = exp(predict(tfm.glm, 
                         newdata = data.frame(
                             maleNo = mNo,
                             femaleNo = fNo,
                             age = 'yearling',
                             yearF = yr,
                             intensity = intV,
                             cwd = 'neg',
                             days = 91
                         ), type = 'response') - 2)

    a.pred = exp(predict(tfm.glm, 
                         newdata = data.frame(
                             maleNo = mNo, 
                             femaleNo = fNo,
                             age = 'adult',
                             yearF = yr,
                             intensity = intV,
                             cwd = 'neg',
                             days = 91
                         ), type = 'response') - 2)
    if (is.null(yl)) {
        yl = c(0, max(c(f.pred, y.pred, a.pred)))
    }
    plot(f.pred / (mNo + fNo) ~ intV, type = 'l', col = 2, 
         ylim = yl,
         xlab = 'Culling Intensity', ylab = 'Mass per Fetus at Apr 1',
         main = paste('Litter: ', mNo, ' male(s), ', fNo, ' female(s)', 
                      sep = ''))
    lines(y.pred / (mNo + fNo) ~ intv, col = 6)
    lines(a.pred / (mNo + fNo) ~ intv, col = 4)
}

Note that the graphs show the mass per fetus rather than total fetal mass on the y-axis, so that they may be directly compared. Also note that the predictions are for the mass at Apr. 1, the latest date in our data set.

In spite of the relatively good fit of the model, there are reasons to be suspicious of these results. The middle left graph predicts that for litter sizes of two, fawns have larger fetuses than yealings or adults in the absence of culling. This seems untenable. Furthermore, it is unclear how an increase in culling intensity could lead to both a decrease in male fetus size and an increase in female fetus size.

As with the fetal size ratio, we return to the total fetal mass in other sections to see if we can better interpret the data.
