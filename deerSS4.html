Litter Sizes

Next we turn to the affect of culling intensity on the litter size, or number of fetuses per pregnancy.

We will consider the same predictors as before. But first, look at the distribution of the response (number of fetuses), which for this analysis we exclude does that were not pregnant.

The response is not normal, and none of the usual transformations will work to normalize the data. But, as these are count data, they mey be sufficiently well-described by a Poisson distribution. Below we compare a simple linear model to a glm with Poisson distributed residuals.

# Because of missing data, the step function fails to reach its conclusion.
# By setting trace = 1 (or simply omitting the trace argument), the output at 
# each iteration is given, so we can manually move the dropped predictors and 
# continue.  This is a slightly imperfect solution, because it means that not 
# exactly the same underlying data are contibuting to the model, and hence 
# AIC values are not directly comparable.  However, given the size of the 
# data set, the affect of the missing data points should be minimal.
litter.full.poiss = glm(fetusNo ~ (intensity + I(intensity^2) + age +
                        yearF + cwd)^2 - age:yearF - yearF:cwd - 
                        intensity:yearF - I(intensity^2):yearF - 
                        yearF - age:cwd - intensity:age - I(intensity^2):age 
                        - intensity:cwd - I(intensity^2):cwd - 
                        intensity:I(intensity^2) - I(intensity^2) - 
                        intensity - cwd, 
                        data = subset(deerF1, fetusNo > 0 & !is.na(fetusNo)),
                        family = poisson)
litter.red.poiss = step(litter.full.poiss, direction = 'both', trace = 0)

As we can see, both the fit and the diagnostics of the glm are much better. If we want to obtain the p value for each categorical value as a whole, we can use the aov (analysis of variance), function. And if we wish to look at pairwise differences, then we use Tukey's Honest Significant Differences to adjust the p value to compensate for the fact that we are making multiple comparisons.

The results of the glm indicate that the age of the mother, sharpshooting intensity, CWD status, and age-intensity interactions are all significant. As we are primarily interested in the effects of the culling, we visualize.

Recalling that glms typically transform the response according to the link function, we must perform the inverse function to get the results back in the original units. Again we use the 'typical' year.

yrCoef = median(c(0, coef(litter.red.gamma)[6:14])) 
yYrCoef = median(coef(litter.red.gamma)[seq(18, 34, 2)])
aYrCoef = median(coef(litter.red.gamma)[seq(19, 35, 2)])

intv = seq(min(deerF1$intensity, na.rm = T), 
           max(deerF1$intensity, na.rm = T), length = 100)

litter.pred = function(int, yrl, ad, dis) {
    pred = 1 + 0.001883*int - 0.0002036*int^2 - 0.3333*yrl - 0.4946*ad +
           yrCoef + 0.04643*dis + 0.0002321*int^2*yrl + 0.0001567*int^2*ad +
           yYrCoef*yrl + aYrCoef*ad
    1 / pred
}

fPred = litter.pred(intv, F, F, dis = F)
yPred = litter.pred(intv, T, F, dis = F)
aPred = litter.pred(intv, F, T, dis = F)

plot(fPred ~ intv, type = 'l', col = 2, 
     ylim = range(fPred, yPred, aPred), xlab = 'Sharpshooting intensity',
     ylab = "Litter size")
lines(yPred ~ intv, col = 6)
lines(aPred ~ intv, col = 4)
legend('topleft', lty = 1, col = c(4, 6, 2), 
       legend = c('Adults', 'Yearlings', 'Fawns'), bty = 'n')


The gamma-response is the best fitting though still not a great model. We have reduced the null deviance of 231.7 down to a residual deviance of 189.1 (Nagelkerke r2 = 0.191), which is better than the competing models but not great, and while the predicted affects on adults and fawns in the graphic above seem reasonable, the predicted effect on fawns–namely that they should have higher prengancy rates than yearlings or adult at the highest culling intensities–seems hard to fathom.

We might instead fit separate models to each age class, or try another approach altogether, as the main difficulty seems to be in accurately representing the response distribution.

At this point our models conflict, and it remains unclear if the culling is significantly affecting litter sizes. Our best fitting model suggests that culling intensity is positively correlated with fawn litter sizes, negatively with yearling litter sizes, and has different affects at different intensities on adult litter sizes. We investigate this question further in the machine learning section, where we investigate classification tree methods.
