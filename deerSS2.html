Description of variables

  • earTag: unique identifier for does
  • sex1, sex2…: sex of each fetus
  • cr1, cr2…: crown-rump length for each fetus
  • wt1, wt2…: mass of each fetus
  • wtTotal, crTotal: total mass or cr of all fetuses in the litter
  • fetusNo: number of fetuses detected
  • wtMean, crMean: mean mass or cr of all fetuses in a litter
  • year: year doe was killed
  • crNo, wtNo: number of cr or wt measurements available in a litter
  • maleNo, femaleNo: number of male or female fetuses in a litter
  • age*: there are several methods for age estimation, only “age3categories” is used, with the categories “fawn”, “yearling”, and “adult”
  • pregnant: 1 if fetuses detected, otherwise 0
  • killDate: date doe was culled (dd-Mon-yy)
  • days: the previous two columns are numeric representations of the date; days is such that it counts days from the beginning of the year (e.g. 1 = Jan 1), regardless of the year
  • township, section: locations (see text)
  • townshipTotalYrs: total number of years that sharpshooting (SS) occurred in the township
  • townshipYTD: number of years that SS occured in the township up to the date in question
  • townshipEffort: cumulative number of does killed in the township to date
  • townshipIntensity: see text
  • section*: same as township stats, but at the section level
  • cwdStatus: indicates if the doe was positive or negative for chronic wasting disease

Fetus Detection

In the early stages of pregnancy, fetuses are small, and may escape detection by the biologists in the field. Here we plot the proportion of deer that are found to be pregnant as a function of days elapsed since the new year. Using non-linear least squares the best sinusoidal curves are fit to each age class. This model was chosen as it reaches an asymptote which can be thought of as estimating the true pregnancy rate once all fetuses are detectable.

# First structure data so that we have the counts not-pregnant and pregnant 
# for each age class
fPregDays.t = table(deer$days[deer$age == 'fawn'], 
                    deer$pregnant[deer$age == 'fawn'])
yPregDays.t = table(deer$days[deer$age == 'yearling'], 
                    deer$pregnant[deer$age == 'yearling'])
aPregDays.t = table(deer$days[deer$age =='adult'],
                    deer$pregnant[deer$age == 'adult'])

# Add an additional column = fraction pregnant at each day
fPregDays.t = cbind(fPregDays.t, 
                    fPregDays.t[,2] / (fPregDays.t[,1] + fPregDays.t[,2]))
colnames(fPregDays.t)[3] = 'FractionPregnant'

yPregDays.t = cbind(yPregDays.t, 
                    yPregDays.t[,2] / (yPregDays.t[,1] + yPregDays.t[,2]))
colnames(yPregDays.t)[3] = 'FractionPregnant'

aPregDays.t = cbind(aPregDays.t, 
                    aPregDays.t[,2] / (aPregDays.t[,1] + aPregDays.t[,2]))
colnames(aPregDays.t)[3] = "FractionPregnant"

fDays = as.numeric(rownames(fPregDays.t))
yDays = as.numeric(rownames(yPregDays.t))
aDays = as.numeric(rownames(aPregDays.t))

# Plot with point sizes scaled to reflect the number of data contributing
plot(fDays, fPregDays.t[, 3], xlim = c(1, 91), ylim = c(0, 1), 
     xlab='Days into the New Year', 
     ylab='Fraction of does with fetuses detected', 
     cex = (fPregDays.t[, 1] + fPregDays.t[, 2]) / 10, 
     pch = 16, col = rgb(1, 0, 0, 0.66))
points(yDays, yPregDays.t[,3], pch = 16, col = rgb(1, 0, 1, 0.66),
       cex = (yPregDays.t[, 1] + yPregDays.t[, 2]) / 10)
points(aDays, aPregDays.t[,3], pch = 16, col = rgb(0, 0, 1, 0.66),
       cex = (aPregDays.t[, 1] + aPregDays.t[, 2]) / 10)

# Fit sigmoid curve: y = L / (1 + C * exp(-k*(x - d))); 
# L: limit (capacity); C = time to L; k = steepness; d = displacement
pregDaysF.nls = nls(fPregDays.t[, 3] ~ 
                    a*(exp(b*fDays - c) / (1+exp(b*fDays-c))),
                    start = list(a = 0.2, b = 1, c = 20),   
                    weights = fPregDays.t[, 1] + fPregDays.t[, 2])
#summary(pregDaysF.nls) 
A = coef(pregDaysF.nls)[1]
B = coef(pregDaysF.nls)[2]
CC = coef(pregDaysF.nls)[3]

xv = seq(0, 91, 0.1)
yv = A*(exp(B*xv - CC) / (1 + exp(B*xv - CC)))
lines(xv, yv, col = 2, lwd = 2)

# Because the last few yearling data have unusually low proportions, if we 
# fit the function as is, we get a graph that begins at the asymptote, and 
# drops to 0 at the far right. To force an increasing fit we remove the 
# smallest number of data points possible (2) from the right.
yPregDays.t2 = yPregDays.t[-c(77:78), ]
pregDaysY.nls = nls(yPregDays.t2[, 3] ~ 
                    a*(exp(b*yDays - c) / (1 + exp(b*yDays - c))), 
                    start = list(a = 0.73, b = 0.08, c = 5),
                    weights = yPregDays.t2[, 1] + yPregDays.t2[, 2])
#summary(pregDaysY.nls) 
A = coef(pregDaysY.nls)[1]
B = coef(pregDaysY.nls)[2]
CC = coef(pregDaysY.nls)[3]
yv = A*(exp(B*xv - CC) / (1 + exp(B*xv - CC)))
lines(xv, yv, col = 6, lwd = 2)

pregDaysA.nls = nls(aPregDays.t[, 3] ~ 
                    a*(exp(b*aDays - c) / (1 + exp(b*aDays - c))), 
                    start = list(a = 0.8, b = 1, c = 19),
                    weights = aPregDays.t[, 1] + aPregDays.t[, 2])
#summary(pregDaysA.nls) 
A = coef(pregDaysA.nls)[1]
B = coef(pregDaysA.nls)[2]
CC = coef(pregDaysA.nls)[3]
yv = A*(exp(B*xv - CC) / (1 + exp(B*xv - CC)))

The 31-day mark (Jan 31; vertical black line) was somewhat arbitrarily chosen as the day after which fetuses become reliably detectable (e.g., where the fitted functions begin to level off). Hence we omit data prior to this date as it may bias the data toward artificially low pregancy rates.

deerF1 = subset(deer, days > 31)

Fetal Size and Growth

Next we examine metrics for the fetus sizes.

The purpose of the above graphics is to demonstrate that the mass has a non-linear relationship with body length. We to create a metric that captures the 'size' of a fetus (e.g., as an indicator of its fitness), but that is constant regardless of its gestation age, so that we can use the same metric for all fetuses regardless. The second graph shows that the log - log relationship between mass is linear, and hence log(mass) / log(crLength) = constant for all gestation ages, though the constant differs for litter size.

We can further investigate this claim by modeling the log(mass) regressed on log(crLength). Here I exemplify with twin litters.

loglog.m  = lm(log(wtMean[fetusNo == 2]) ~ log(crMean[fetusNo == 2]), 
               data = deerF1)
summary(loglog.m)
